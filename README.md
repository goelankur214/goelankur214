## 🎮 Featured Project: Slapstick Sky Safari

An ad-free, secure keyboard-smashing and screen-tapping sandbox game engineered specifically for infants and toddlers. It serves as an early sensory tool that gracefully handles unstructured input.

* **🔗 Live Web App:** [play slapsticksky.xyz](https://slapsticksky.xyz)
* **💻 Tech Stack:** HTML5 Canvas, Vanilla JavaScript (ES6+), Web Audio API, Vercel Edge Hosting.

### 💡 Core Architecture & Technical Highlights
* **Procedural Audio Synthesis:** Bypasses heavy static audio asset files (.mp3/.wav) by generating live sound waves (oscillators, custom gain envelopes) programmatically using the browser's Web Audio API. 
* **Harmonic Scale Mapping:** Synthesizes sound frequencies constrained to a Pentatonic scale (C, D, E, G, A) to ensure randomized toddler keyboard smashing outputs melodic, parent-friendly chimes.
* **Dual-Age State Matrix:** Dynamically shifts context based on user selection. Spawns high-contrast floating emojis on a neon twilight sky for infants (Ages 1-2), or converts keystrokes into massive alpha-burst text particles on a bright daytime theme for older kids (Ages 3-4).
* **Robust Client Lockout Security:** Implements a global event intercept (`event.preventDefault()`) to neutralize standard browser hotkeys and developer tools, paired with an asynchronous 3-second long-press timer (`onmousedown`/`ontouchstart`) to secure the hidden parent control module.

## 🛠️ Featured Project: RadiatorSB

A high-performance, mobile-first B2B/B2C automated e-commerce and parts procurement engine tailored for regional cooling parts distributors and local automotive repair workshops.

* **🔗 Live Platform:** [visit radiatorsb.us](https://www.radiatorsb.us/)
* **💻 Tech Stack:** Semantic HTML5, Vanilla JavaScript (ES6+), CSS3 Grid/Flexbox, Sanity.io (Headless CMS), Make.com (Serverless Automation Webhooks), Vercel Global Edge CDN.

### 💡 Core Architecture & Technical Highlights
* **Decoupled Headless CMS Pipeline:** Integrated **Sanity.io** via a CDN-cached API query pipeline to dynamically stream, filter and render real-time parts catalogs and specifications without structural re-renders.
* **60fps Client Performance Tuning:** Eliminated browser layout reflows by utilizing GPU-accelerated CSS `transform: translateX` layers for menu transitions. Consolidated window scroll triggers into a single listener throttled natively via `requestAnimationFrame` with `{ passive: true }` flags.
* **Automated Serverless Workflows:** Tied the primary transaction and quotation funnels directly to **Make.com** serverless webhooks, safely parsing metadata and instantly routing leads to internal regional dispatch channels.
* **WCAG Compliant Accessibility (A11y):** Implemented strict ARIA standard maps (`aria-hidden`, `aria-expanded`), runtime keyboard focus traps (allowing `Escape` key close states) and strict 44px layout touch boundaries.
